OpenMiuiFTP
===========
####运行截图
![Alt text](/path/to/img.jpg)
##功能：
####pc端自动打开MIUI远程管理FTP地址

* 端口写死是默认的2121

* ip地址可以自适配

* 扫描范围为：

    (路由器ip地址前3段) XXX.XXX.XXX.100 ~ XXX.XXX.XXX.119

####为什么不扫描更大范围？：
    大多数家用路由器DHCP服务器 分配局域网的地址池是 从192.168.X.100开始，到192.168.X.199结束。
    100到119可以连接20个设备了，相信一般人家里路由器不会连这么多设备吧。
